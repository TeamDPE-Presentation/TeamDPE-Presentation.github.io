The Problem

Agile relies on constant communication to ensure its success.

There are many enterprise-wide tools to help developers and managers track progress.

The Solution

“A lot of valuable information in a single place”

DASH is a tool for both developers and managers to keep track on their team’s progress

Our Stack

Back End

  • Java 1.7+
  • Spring (SpringBoot and SpringMVC)
  • Amazon Web Services

Front End

  • Javascript, jQuery, AngularJS
  • HTML5
  • CSS, Bootstrap, Sass and LESS

Dash B.I (Before Interns)

State

  • Pulled from Hudson, VersionOne, Subversion and SonarQube
  • Templated UI from Bootstrap, few custom css changes
  • Widgets were stagnant and non responsive

The New Dash

  • Jenkins, Git, Splunk and Jira data aggregation added
  • New minimalistic, flat design. Reorganized informations to separate widgets for clarity
  • Improved widgeting system with layout and theming options
  • Now deployed on AWS
